How to use:



Please make sure your OS is win7 64bit and you have already configure python cgi already and add MySQL-python module to your computer. The database is imported into the phpmyadmin and named as csv_db.

Then put babynames.html and cgi-bin folder which includes the babynames.py to your root such as www folder. Note that cgi-bin is your cgi directory.

Go to your browser and type localhost/babynames.html

If have any question, please contact me.
